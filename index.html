HTML

  • The head element contains information about webpage.
  • This information is called metadata and is only viewed by prorammers.
  • Elements that are in the head section include title, which shows the name of the website in the tab at the top of the browser.
  • Another element is meta which describes the metadata within a HTML document e.g. character set and viewport.
  • The body element represents the visible content shown to the user.
  • It is also called parent element as it contains sub elements within it, child elements.
  • There is h element used for heading different levels of headers as well as p tag used to indicate a paragraph or a block of text.
  • Other elements that can be used for listing both non-organised and organised list are ol, ul and li.
  • img element is used to display images on a website and br element is used to create line break or empty space.

Git

  • git status: checks what branch we are on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch name: switches to said branch
  • mkdir name: creates a new directory for a new work
  • git add -A: adds new changes to whichever branch
  • git commit: commits changes to the history
  • git commit -m "text": adds a comment to explain changes for other users in the project
  • git rm file: removes from both working directory and repository.
