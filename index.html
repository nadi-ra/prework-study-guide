HTML

  • The head element contains about the webpage.
  • This information is known as meta data.
  • In this section exits the meta tag which used to describe a website like characters and viewpoints.
  • Another tag is title which shows up on the tab at the top of the browser.
  • The body element represents the visible content shown to the user.
  • Its considered a parent elements as it holds other elements known as child elements.
  • The p element is used for paragraphs or block of texts and the img tag is used to display images.
  • The tags ol, ul and li can be used to lists both organised and unorganised texts.
  • The h tag is used to make a heading and has different levels of heading.

CSS

  • There are three ways to style a webpage; inline CSS, internal CSS style sheet and external CSS style sheet.
  • Inline CSS is used in the HTML files and it styles specific elements such as img and h using attributes.
  • Internal CSS style sheet is found in the head element at the top of the file.
  • The style element is used to design multiple tags using CSS rules.
  • The most popular option is external CSS style sheet because it allows developers to have seperate files for all CSS rules.
  • The link element is used to connect between external style sheet and the code file.
  • A CSS rule contains a selector which defines the element/attribute to which rules/decleration will apply.
  • A margin indicates how much space is wanted outside an element.
  • A padding indicates how much space we want around the content inside an element.
